# Cinnamon's Kitchen Website

This is a simple front end website for a fictional restaurant. 

You can [view the live website here](https://isabels-t.github.io/cinnamon/)

---

## View Optimization
* **Laptop View Only**: This website is currently designed and optimized for laptop and desktop screens only. 
* Mobile View: The layout has not yet been adjusted for mobile phones or smaller tablet screens.

## How It Was Built
This is a **front end only** project. There are no databases or backend servers. 
* HTML - Website structure and content
* CSS - Colors, fonts, and layout
* JavaScript - Menu interactions and basic features

## Project History
* Created in **2023** using **Visual Studio Code (VS Code)**.
* Kept on a local computer and recently uploaded to GitHub as a portfolio project.
